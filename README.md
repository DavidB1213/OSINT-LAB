# 🔎 OSINT-LAB

OSINT-LAB es un entorno de investigación OSINT basado en Linux, diseñado para centralizar, organizar y automatizar el uso de herramientas de inteligencia en fuentes abiertas.

---

## 🎯 Objetivo

Reducir la complejidad en la ejecución de herramientas OSINT mediante una capa de abstracción basada en scripts lanzadores.

---

## 🧠 Características principales

- Sistema modular por categorías
- Automatización mediante Bash
- Integración con entorno gráfico (XFCE)
- Accesos directos (.desktop)
- Flujo guiado para el analista

---

## 🧩 Arquitectura

El sistema se divide en:

- `/OSINT/` → herramientas y scripts
- `/Desktop/OSINT-LAB/` → interfaz visual

---

## 🚀 Ejecución

```bash
bash main_launcher.sh
```
